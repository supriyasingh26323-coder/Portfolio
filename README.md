# Supriya Singh — Personal Portfolio

A responsive personal portfolio website showcasing my projects, skills, and experience as an aspiring software developer.

🔗 **Live Site:** https://supriyasingh26323-coder.github.io/portfolio/

## About

This portfolio is built with a code-editor/terminal-inspired design — reflecting my background as a developer. It highlights my technical skills, academic projects, certifications, and education.

## Features

- Terminal-style animated hero introduction
- Responsive navbar with smooth scrolling and active-section highlighting
- Mobile-friendly hamburger menu
- Skills section with technology icons
- Project showcase with GitHub repository links
- Education & certifications timeline
- Downloadable resume
- Scroll-triggered section animations
- Fully responsive across devices

## Tech Stack

- HTML5
- CSS3 (custom properties, Grid, Flexbox)
- Vanilla JavaScript
- [Font Awesome](https://fontawesome.com/) for icons

## Run Locally

1. Clone the repository
   ```
   git clone https://github.com/supriyasingh26323-coder/portfolio.git
   ```
2. Open `index.html` in your browser — no build step required.
